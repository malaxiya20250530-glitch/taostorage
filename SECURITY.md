# 🛡️ TaoStorage 安全策略

## 报告漏洞

发现安全漏洞请发送邮件至 security@tao.storage（暂不可用，先用 GitHub Issues 加 `[SECURITY]` 前缀）。

## 安全特性

### 传输层
- **TLS 1.3**：所有 HTTP/WebSocket 通信
- **Noise 协议**：libp2p 传输加密
- **流量混淆**：防流量分析

### 存储层
- **内容寻址**：SHA256 哈希确保数据完整性
- **端到端加密**：x25519 + ChaCha20Poly1305
- **零知识证明**：可验证存储但不泄露内容

### 网络层
- **Kademlia DHT**：去中心化路由
- **信誉系统**：防止 Sybil 攻击
- **存储证明**：节点必须证明确实存储了数据

## 最佳实践

1. **永远不要** 将私钥/种子短语分享给任何人
2. 使用 `tao export` 定期备份数据
3. 加入网络时验证引导节点身份
4. 敏感数据建议先自行加密再存储
